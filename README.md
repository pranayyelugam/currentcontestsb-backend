

### Instructions

#### Installation

```bash
# Setup a virtual env
$ virtualenv env

# Activate it
$ source env/bin/activate

# Install dependencies
(env) $ pip install -r requirements.txt
```

#### Running the app

```bash
(env) $ python route.py
```
